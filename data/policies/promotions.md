# Promotions and Coupons Policy

## 1. Coupon Usage
- Only one coupon code can be used per order.
- Coupons cannot be applied to "Final Sale" or "Daily Deal" items.
- Coupons cannot be applied after an order has been placed.

## 2. Promotional Refunds
If an item was purchased as part of a "Buy One, Get One" (BOGO) promotion, the refund value will be adjusted.
- If only one item is returned, the refund will be the price paid minus the full value of the item kept.

## 3. Expiration
Promotional credits and coupons expire on the date specified. No extensions will be granted.
