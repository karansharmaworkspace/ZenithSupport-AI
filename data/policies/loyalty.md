# Purple Prime Membership and Loyalty

## 1. Prime Benefits
Purple Prime members receive exclusive "Concierge Return Service."
- **Extended Window:** Members get 60 days to return items (vs. 30 days for standard accounts).
- **Free Home Pickup:** For select urban regions, we offer free home pickup for bulky return items.

## 2. Loyalty Points Redemption
When an order is placed using Purple Points:
- **Refund logic:** Points are always returned first. If an order cost $100 ($20 in points and $80 in cash), a partial refund of $50 will return $20 in points and $30 in cash.
- **Point Expiration:** Refunded points retain their original expiration date. 

## 3. Promotional BOGO (Buy-One-Get-One) Deals
For "Buy One, Get One Free" or "Buy One, Get One 50% Off" promotions:
- **Partial Returns:** If you return only one item from a BOGO pair, you will be refunded the difference between the full price of the item you kept and the total price you paid. 
- **Example:** If you buy two $20 shirts for $20 (BOGO Free) and return one, you will receive $0 refund because you kept one shirt worth the $20 you paid.
